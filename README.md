# 🏡 House Price Prediction with Python

## Project Overview
This project analyzes King County housing data using Python, data analysis, and machine learning techniques to predict house prices.

## Key Objectives
- Data Cleaning & Wrangling
- Exploratory Data Analysis (EDA)
- Feature Selection
- Linear Regression
- Ridge Regression
- Polynomial Ridge Regression

## Key Insights
- Waterfront homes had ~3.1X higher median prices than non-waterfront homes
- Sqft living was a far stronger predictor than longitude
- Polynomial Ridge Regression improved R² from 0.648 to 0.700

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

## Final Model Performance
**Polynomial Ridge Regression: R² = 0.700**
